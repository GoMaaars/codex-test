# Codex Task Routing

This repository is used as a lightweight task handoff hub.

When Codex is opened in this repository:

1. Read `tasks/CODEX_ACTIVE_TASK.md` first.
2. Execute the task end-to-end unless blocked by credentials or an irreversible external action.
3. Do not expand scope beyond the acceptance criteria.
4. Record progress and blockers in `tasks/CODEX_RESULT.md`.
5. For tasks that create a new repository, complete the work in the new repository and push all commits there.
6. Never add investment advice, trading signals, buy/sell recommendations, or predictive claims to market-data products unless explicitly authorized in a later task.
